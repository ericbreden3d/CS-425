Documentation:

Use sliders to adjust kaleidoiscope parameters. Each parameter change will result in generation of new randomized triangle points that will be mirrored to form the kaleidoscope.

Parameters:

N: the number of angular subdivisions to mirror around
Layers: the number of radial subdivisions to generate (each layer has its own randomized values)
Angle Mult: multiplier applied to point angle values for interesting results that allow overlapping triangles
